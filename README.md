# 🐾 Sistema de Clínica Veterinaria

Aplicación desarrollada en **Java** para la gestión integral de una clínica veterinaria.  
Permite administrar mascotas, dueños, citas médicas y tratamientos de forma eficiente.

---

## 🚀 Descripción

Este sistema está diseñado para cubrir las necesidades básicas de una clínica veterinaria, facilitando:

- Registro de mascotas 🐶🐱  
- Gestión de dueños 👤  
- Control de citas 📅  
- Historial clínico 🩺  
- Administración general del sistema
